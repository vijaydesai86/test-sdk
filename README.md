# Test SDK

## Overview
This repository is a Next.js web application designed for stock research. It serves various report types including:

- **Stock Research**  
- **Comparison Reports**  
- **Deep Research**  
- **Watchlist Reports (Daily)**

### Features
- Support for multiple report types to accommodate diverse user needs.
- Integration with Supabase for optional data persistence.
- Environment variables for configuration.

## Getting Started
To run the application locally, follow these steps:
1. Clone the repository:  
   `git clone https://github.com/vijaydesai86/test-sdk.git`
2. Navigate into the project directory:  
   `cd test-sdk`
3. Install dependencies:  
   `npm install`
4. Set up your environment variables (see `.env.example` for guidance).
5. Start the development server:  
   `npm run dev`

Visit `http://localhost:3000` to view the app in your browser.

## Deployment
To deploy the application on Vercel:
1. Push your code to GitHub.
2. Create a Vercel account and link your GitHub repository.
3. Follow the instructions to set up and deploy your application.
